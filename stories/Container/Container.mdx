`Container` コンポーネントは、コンテンツを中央に配置し、最大幅を制限するために使用されます。

## 使い方

```tsx
import { Container } from "@/components/Container/Container";

const Demo = () => (
  <Container size="md" p={20}>
    中央に配置されるコンテンツ
  </Container>
);
```

<Canvas of={ContainerStories.Default} />

## 特徴

- **中央配置**: `mx="auto"` がデフォルトで適用され、コンテンツが常に中央に配置されます。
- **レスポンシブなサイズ**: `xs`, `sm`, `md`, `lg`, `xl` のプリセットサイズ、または数値/文字列でのカスタムサイズ指定が可能です。
- **フル幅オプション**: `fluid` プロパティを使用することで、最大幅を制限せずに 100% の幅で表示できます。
