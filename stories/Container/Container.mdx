`Container` コンポーネントは、コンテンツを中央に配置し、最大幅を制限するために使用されます。

## 使い方

```tsx
import { Container } from "@/components/Container/Container";

const Demo = () => (
  <Container size="md" p={20}>
    中央に配置されるコンテンツ
  </Container>
);
```

## 特徴

<table>
  <thead>
    <tr>
      <th>特徴</th>
      <th>説明</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>中央配置</strong></td>
      <td><code>mx="auto"</code> がデフォルトで適用され、コンテンツが常に中央に配置されます。</td>
    </tr>
    <tr>
      <td><strong>レスポンシブなサイズ</strong></td>
      <td><code>xs</code>, <code>sm</code>, <code>md</code>, <code>lg</code>, <code>xl</code> のプリセットサイズ、または数値/文字列でのカスタムサイズ指定が可能です。</td>
    </tr>
    <tr>
      <td><strong>フル幅オプション</strong></td>
      <td><code>fluid</code> プロパティを使用することで、最大幅を制限せずに 100% の幅で表示できます。</td>
    </tr>
  </tbody>
</table>
