# InteractiveArea

`InteractiveArea`は、`Dropzone`や`EmptyState`などのコンポーネントの基盤となる共通コンポーネントです。
一貫した枠線、背景、余白、およびホバーやドラッグ状態のインタラクションを提供します。

## Usage

```tsx
import { InteractiveArea } from "./components/InteractiveArea/InteractiveArea";
import { Icon } from "./components/Icon/Icon";

const MyArea = () => (
  <InteractiveArea
    title="データの準備ができていません"
    description="新しいプロジェクトを作成して開始してください。"
    icon={<Icon name="ProjectIcon" size="large" />}
    variant="dashed"
    isClickable
  />
);
```

## Relation with ContextMenu

`ContextMenu`コンポーネント自体は `InteractiveArea` を内部で使用していません。その理由は以下の通りです：

1.  **汎用性の維持**: `ContextMenu` はテキストや単一のボタンなど、あらゆる要素を右クリックの対象（トリガー）としてラップする責務を持ちます。
2.  **スタイルの分離**: `InteractiveArea` は特定の余白や枠線を持つ「エリア」としての見た目を提供しますが、全ての右クリックメニューにこのスタイルが必要なわけではありません。

右クリック可能なインタラクティブエリアを作成する場合は、以下のように組み合わせて使用します：

```tsx
<ContextMenu menu={<MyMenu />}>
  <InteractiveArea title="右クリックエリア" variant="solid" />
</ContextMenu>
```
