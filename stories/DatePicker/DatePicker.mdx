## カレンダー選択フロー

- 1回目のクリックで開始日を選択します。
- 2回目のクリックで終了日を選択します。
- 選択中は開始日〜終了日の間がハイライト表示されます。
- 開始日より前の日付を終了日として選択した場合、自動的に入れ替わります。

## <T k="doc_datepicker_ts" />

<p><T k="doc_datepicker_ts_desc" /></p>

```tsx
import { DatePicker } from "./DatePicker";
import { T } from "../T";

<DatePicker
  value={new Date()}
  onChange={(date: Date | null) => {
    if (date) {
      console.log(date.toISOString());
    }
  }}
  format="YYYY-MM-DD"
  clearable
/>;
```
