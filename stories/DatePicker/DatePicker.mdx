## <T k="doc.datepicker_ts" />
<p><T k="doc.datepicker_ts_desc" /></p>

```tsx
<DatePicker
  value={new Date()}
  onChange={(date: Date | null) => {
    if (date) {
      console.log(date.toISOString());
    }
  }}
  format="YYYY-MM-DD"
  clearable
/>;
```
