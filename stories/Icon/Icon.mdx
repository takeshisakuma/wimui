アプリケーション全体で使用されるアイコンコンポーネント

## Iconを独立したコンポーネントにする利点

- すべてのアイコンが一元管理できるのでデザインの一貫性を保てる
- アイコンのファイル形式やスタイリングなどをアイコンコンポーネントにカプセル化するので変更がしやすくなる
- ボタンだけでなく、リストアイテム、フォームの入力フィールド、ステータス表示など、さまざまなコンポーネントで同じアイコンを簡単に再利用できる

## Icon のアニメーションと Loader コンポーネントの使い分け

Icon コンポーネントでは `LoadingIcon` や `SpinnerIcon` を指定すると、**360度回転アニメーション（`wim-rotate`）**が自動的に適用されます。一方で、別途 **Loader コンポーネント** にも読み込み中を示すアニメーションがあります。これらは用途が異なるため、適切に使い分けてください。
