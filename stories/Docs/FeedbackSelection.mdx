# フィードバック・通知コンポーネントの使い分け

システムの状態変化やユーザーの操作結果を伝える際、メッセージの重要度や持続性に応じてコンポーネントを選択します。

## ステータスの意味合い

フィードバックコンポーネントは、色のセマンティクスを活用して情報を伝えます。

- **Success (Green)**: 操作が成功した、正常に完了した。
- **Info (Blue)**: 補足的な情報、中立的な通知。
- **Warning (Yellow/Orange)**: 注意が必要。
- **Error (Red)**: 致命的な問題、操作の拒否。
