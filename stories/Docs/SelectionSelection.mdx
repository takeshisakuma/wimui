# 選択系コンポーネントの使い分け

ユーザーに値を選択させる際、選択肢の数や入力の性質に応じて最適な形式を選択します。

## 選択のフローチャート

1. **オン/オフの切り替えか？**
    - 即時反映なら ➔ **Switch**
    - 送信が必要なら ➔ **Checkbox**
2. **1つだけ選ぶか、複数選ぶか？**
    - 複数選ぶ ➔ **Checkbox Group**
    - 1つだけ選ぶ ➔ 次へ
3. **選択肢の数は？**
    - 少なめ (2-5) ➔ **Radio** または **Segmented Control**
    - 普通 (5-20) ➔ **Select**
    - 多い (20-) ➔ **Combobox**
