ToggleGroup components allow users to select one or multiple options from a set of buttons. It's often used for formatting options, alignment, or filtering where a button-like interface is preferred over checkboxes or radio buttons.

## When to use

- When you need a button-like interface for selection.
- For toolbars (e.g., text formatting: Bold, Italic, Underline).
- When binary toggle states (On / Off) are grouped together.
