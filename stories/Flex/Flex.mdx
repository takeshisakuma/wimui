`Flex` は、CSS Flexboxを利用してレイアウトを構築するための基本的なコンポーネントです。
要素の整列、方向、間隔などをプロパティ経由で直感的に制御できます。

## 使い方

```tsx
import { Flex } from "@/components/Flex/Flex";

const Demo = () => (
  <Flex direction="row" align="center" justify="between" gap={16}>
    <div>アイテム 1</div>
    <div>アイテム 2</div>
    <div>アイテム 3</div>
  </Flex>
);
```

## 垂直方向の並び (Column)

`direction="column"` を指定することで、要素を垂直に並べることができます。

<Canvas of={FlexStories.Column} />

## コンテンツの分散配置 (Justify Content)

`justify` プロパティを使用すると、メイン軸（rowの場合は横方向）に沿った要素の配置方法を指定できます。

<Canvas of={FlexStories.JustifyBetween} />

## 折り返し (Wrap)

`wrap="wrap"` を指定すると、親コンテナの幅を超えた要素が自動的に次の行に折り返されます。
