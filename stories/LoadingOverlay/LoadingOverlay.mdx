# LoadingOverlay

A component that overlays the entire screen or a specific element to display a loading state with customizable loaders, backdrops, and messages.

## When To Use

- When you need to block user interaction during asynchronous operations like data fetching or form submission.
- When you want to provide visual feedback that a process is ongoing.
- When you need a full-screen loading indicator for application initialization.
- When you want to overlay a specific section of the page during partial updates.

## Accessibility

- The component includes `role="status"` and `aria-live="polite"` for screen reader announcements.
- The `aria-busy="true"` attribute indicates that the content is loading.
- Ensure that loading messages are descriptive and helpful for users relying on assistive technologies.
