# Popover

<p style={{ margin: "0 0 16px 0", fontSize: "18px", lineHeight: "28px", color: "#6c757d" }}>
  <T k="doc_popover_title" />
</p>

ボタンクリックでトリガーされ、リッチコンテンツをオーバーレイ表示します。

**Note:** `Dialog`コンポーネントと同様にUIをシンプルに保つため、デフォルトでは右上の閉じるボタンを配置していません。ポップオーバーはコンテンツエリア外のクリックやEscapeキーで簡単に閉じることができます。明示的な閉じるアクションが必要な場合は、コンテンツエリア内にボタンを追加してください。

## Usage

```tsx
import { Popover, PopoverContent, PopoverTrigger, PopoverClose } from "@/components/Popover/Popover";

<Popover>
  <PopoverTrigger>Open</PopoverTrigger>
  <PopoverContent>
    Place content for the popover here.
  </PopoverContent>
</Popover>
```
