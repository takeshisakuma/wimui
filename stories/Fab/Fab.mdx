# Fab (Floating Action Button)

Fab（フローティングアクションボタン）は、画面の主要なアクションを促すためのボタン
画面の最前面に固定して配置されることが多く、円形の形状が特徴
「これさえやればOK」というアクション1つだけなので、プライマリのみを用意
Buttonと異なり、状態としてのloadingとdisabledは用意していない

## 特徴
- 画面の主要なアクション(作成、追加、共有など)に使用
- 固定位置(Fixed)またはインライン(Static)で配置
- ラベルを追加して「拡張FAB」として使用することも可能

### 拡張FAB (Extended)

<Canvas of={FabStories.Extended} />

## 配置 (Position)
`position` プロップを使用して、画面の四隅または下部中央に固定配置

- `bottom-right` (デフォルトの固定位置)
- `bottom-left`
- `bottom-center`
- `top-right`
- `top-left`
- `static` (通常フローの中での配置)
