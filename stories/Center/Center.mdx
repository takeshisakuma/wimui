`Center` コンポーネントは、子要素を垂直および水平方向の中央に配置するために使用されます。

## 使い方

```tsx
import { Center } from "@/components/Center/Center";

const Demo = () => (
  <Center h={200} bg="#f0f0f0">
    中央に配置されるコンテンツ
  </Center>
);
```

## 特徴

<table>
  <thead>
    <tr>
      <th>特徴</th>
      <th>説明</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>完全な中央配置</strong></td>
      <td><code>align-items: center</code> と <code>justify-content: center</code> がデフォルトで適用されます。</td>
    </tr>
    <tr>
      <td><strong>インラインモード</strong></td>
      <td><code>inline</code> プロパティを使用することで、<code>inline-flex</code> として動作させることができます。</td>
    </tr>
    <tr>
      <td><strong>柔軟なサイズ</strong></td>
      <td><code>Box</code> や <code>Flex</code> のプロパティを継承しているため、高さ (<code>h</code>) や幅 (<code>w</code>) も簡単に指定できます。</td>
    </tr>
  </tbody>
</table>
