# TimePicker

<p style={{ margin: "0 0 16px 0", fontSize: "18px", lineHeight: "28px", color: "#6c757d" }}>

ユーザーが時間を選択するためのコンポーネントです。ネイティブの `input type="time"` をラップしてスタイルを調整しています。

## Usage

```tsx
import { TimePicker } from "./TimePicker";

<TimePicker />
```
