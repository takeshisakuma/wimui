# MultiSelect

ユーザーが定義済みの選択肢から複数を選択するためのドロップダウンメニューです。

## Usage

```jsx
import { MultiSelect } from "./MultiSelect";

const options = [
  { label: "Apple", value: "apple" },
  { label: "Banana", value: "banana" },
  { label: "Orange", value: "orange" },
];

<MultiSelect 
    options={options} 
    onChange={(vals) => console.log(vals)} // returns string[]
/>
```
