# TabNavigation

<T k="doc_tab_navigation_title" />

TabNavigationは、異なるビューやコンテキスト間を移動するためのリンクセットを提供します。
`Tabs` コンポーネントとは異なり、主にページレベルのナビゲーションやフィルタリング（URL遷移を伴うものなど）を想定して設計されています。

## インポート

```jsx
import TabNavigation from "./TabNavigation";
```

## 基本的な使用方法

デフォルトのスタイルは `underline` です。これは、二次的なページナビゲーションとして一般的に使用されます。

<Canvas of={TabNavigationStories.Default} />

## バリエーション

### Pill (カプセル型)

Pillスタイルは、リストのフィルタリングや、状態の切り替えなど、コンテキストを大きく変更しない操作に適しています。

<Canvas of={TabNavigationStories.Pills} />

### Contained (コンテナ型)

Containedスタイルは、セグメントコントロールに似ており、データの表示形式（例：グラフの時間軸の切り替えなど）を変更する場合によく使用されます。

<Canvas of={TabNavigationStories.Contained} />

`icon` プロパティを使用して、各アイテムにアイコンを追加できます。
