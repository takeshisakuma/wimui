ナビゲーションメニューやアクションリストを表示するための汎用的なメニューコンポーネントです。
サブメニュー、グルーピング、複数の表示モードをサポートしています。

## Usage

```tsx
import { Menu, MenuItem, SubMenu, MenuItemGroup, MenuDivider } from "wimui";
import { T } from "../T";

<Menu>
  <MenuItem icon={<Icon name="CheckIcon" />}>Home</MenuItem>
  <MenuItem>About</MenuItem>
  <SubMenu itemKey="services" title="Services">
    <MenuItem>Service 1</MenuItem>
    <MenuItem>Service 2</MenuItem>
  </SubMenu>
</Menu>;
```

## Components

### Menu

ルートコンポーネント。メニューの表示モードや開閉状態を管理します。

Props:

- `mode`: メニューの表示モード (`"vertical"` | `"horizontal"` | `"inline"`)
- `defaultOpenKeys`: デフォルトで開いているサブメニューのキー配列

### MenuItem

メニューの個別項目です。クリックイベントやアイコンを設定できます。

### SubMenu

展開可能なサブメニューです。`itemKey` プロパティで一意のキーを指定する必要があります。

### MenuItemGroup

メニュー項目をグループ化し、タイトルを表示します。

### MenuDivider

メニュー項目間に区切り線を挿入します。

## Accessibility

- キーボードナビゲーション対応（Tab キーでフォーカス移動）
- `role="menu"` と `role="menuitem"` を適切に設定
- `aria-haspopup` と `aria-expanded` でサブメニューの状態を通知
- disabled 状態の項目には `aria-disabled` を設定
