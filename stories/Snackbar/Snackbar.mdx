# Snackbar

スナックバーは、ユーザーに短いメッセージを通知するために使用されます。通常、画面の下部に表示され、アクションボタンを含めることができます。トーストよりも目立たない、一時的な通知に適しています。

## 便利な使い方 (Provider & Hook)

`SnackbarProvider` と `useSnackbar` フックを使用すると、アプリケーションのどこからでも簡単にスナックバーを表示できます。

```tsx
import { SnackbarProvider, useSnackbar } from './Snackbar';

const App = () => (
  <SnackbarProvider>
    <MyComponent />
  </SnackbarProvider>
);

const MyComponent = () => {
  const { showMessage } = useSnackbar();

  return (
    <button onClick={() => showMessage({ 
      message: "保存されました", 
      variant: "success",
      actionLabel: "閉じる"
    })}>
      スナックバーを表示
    </button>
  );
};
```
