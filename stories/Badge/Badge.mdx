# Badge

バッジコンポーネントは、ステータス、通知数、または短いラベルを表示するために使用されます。

## 基本的な使い方

<Canvas of={BadgeStories.Primary} />

## バリエーション

`variant` プロパティを使用して、バッジのスタイルを変更できます。

### Destructive

<Canvas of={BadgeStories.Error} />

### Outline

<Canvas of={BadgeStories.Outline} />

### Secondary

<Canvas of={BadgeStories.Secondary} />

### Icon with Text

`icon` プロパティを使用して、バッジにアイコンを追加できます。

<Canvas of={BadgeStories.WithIcon} />

### Icon Only

`content` プロパティを省略し、`icon` のみを指定することで、アイコンのみのバッジを表示できます。

<Canvas of={BadgeStories.IconOnly} />
