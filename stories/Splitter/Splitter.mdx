## <T k="doc.usage" />

```tsx
import { Splitter } from "wimui";

<Splitter orientation="horizontal">
  <Splitter.Panel defaultSize={30}>Left Content</Splitter.Panel>
  <Splitter.Handle />
  <Splitter.Panel defaultSize={70}>Right Content</Splitter.Panel>
</Splitter>;
```
