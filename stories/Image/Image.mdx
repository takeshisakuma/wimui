`Image` コンポーネントは、画像を洗練された形で表示するためのラッパーコンポーネントです。
レスポンシブなサイズ調整、角丸、影、枠線、キャプションなどの機能を備えています。

## 特徴

- アスペクト比の制御: `fit` プロパティ（`object-fit`）により、画像の収まり方を調整可能。
- スタイリング: `radius`、`shadow`、`border` プロパティで外観を簡単に変更。
- キャプション付き: `caption` プロパティで画像の下に説明文を追加。

## Usage

```tsx
import { Image } from "@/components/Image/Image";

<Image
  src="path/to/image.webp"
  alt="説明文"
  width={400}
  radius="medium"
  shadow
  caption="サンプルの画像です"
/>;
```
