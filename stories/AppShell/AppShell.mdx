# AppShell

A flexible application shell component that provides a complete layout structure for modern web applications. AppShell combines header, sidebar, main content, footer, and navbar areas into a cohesive layout system.

## Overview

The AppShell component is designed to be the foundation of your application's layout. It handles the positioning and organization of major UI sections, allowing you to focus on building your application's features.

## Composition Pattern

AppShell also provides sub-components for more granular control:

```tsx
<AppShell>
  <AppShell.Header>{/* Header content */}</AppShell.Header>
  <AppShell.Sidebar>{/* Sidebar content */}</AppShell.Sidebar>
  <AppShell.Main>{/* Main content */}</AppShell.Main>
  <AppShell.Footer>{/* Footer content */}</AppShell.Footer>
  <AppShell.Navbar>{/* Navbar content */}</AppShell.Navbar>
</AppShell>
```

## Accessibility

- Uses semantic HTML5 elements (`<header>`, `<aside>`, `<main>`, `<footer>`)
- Proper landmark regions for screen readers
- Keyboard navigation support
- Respects user's motion preferences
- Maintains focus management across sections
