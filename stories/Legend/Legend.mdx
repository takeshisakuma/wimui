# Legend

`Legend` コンポーネントは、`Fieldset` 内でグループのタイトルを表示するために使用されます。

## 使い方

```tsx
import { Fieldset, Legend } from "wimui";
import { T } from "../T";

<Fieldset>
    <Legend>グループのタイトル</Legend>
    {/* フォーム要素 */}
</Fieldset>
```
