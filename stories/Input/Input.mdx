## バリエーション

`variant` プロパティでスタイルを変更できます。

### Outline

枠線がある標準的なスタイルです（デフォルトと同じですが、明示的に指定する場合に使用）。

<Canvas of={InputStories.Outline} />

### Ghost

背景色がないシンプルなスタイルです。
通常時は入力フィールドであることを強調せず、ホバー時やフォーカス時のみ視覚的なフィードバックを提供します。

**使用例:**
- 詳細画面のタイトルなど、表示がメインで編集も可能な「Click-to-Edit」UI
- リストやテーブル内など、入力項目が多数並ぶ画面でのノイズ低減
- 検索バーなど、背景に溶け込ませたい場合

**インタラクション:**
- **未入力時:** プレースホルダーで入力エリアであることを示唆
- **ホバー時:** 薄い背景色が表示され、操作可能であることを通知
- **フォーカス時:** プライマリーカラーのボーダーが表示され、入力状態であることを明示
