Navbarは、ページの上部に配置されるレスポンシブなナビゲーションヘッダーです。

> 📘 Guide: See [Guides / App Layout](?path=/docs/guides-app-layout--docs) for layout patterns.
> ブランドロゴ、ナビゲーションリンク、アクションボタンなどを配置するためのコンテナとして機能します。

## インポート

```jsx
import Navbar from "@/components/Navbar/Navbar";
```

## 基本的な使用方法

`Navbar` コンポーネントの中に `Navbar.Brand`、`Navbar.Content` などを配置して使用します。

```jsx
<Navbar bordered>
  <Navbar.Brand>Brand</Navbar.Brand>
  <Navbar.Content>
    <Navbar.Link href="#" active>
      ホーム
    </Navbar.Link>
    <Navbar.Link href="#">概要</Navbar.Link>
    <Navbar.Link href="#">お問い合わせ</Navbar.Link>
  </Navbar.Content>
</Navbar>
```

## グラスモーフィズム (Glassmorphism)

`glass` プロパティを使用すると、背景にぼかし効果（すりガラス効果）を適用できます。
カラフルな背景画像やグラデーションの上に固定ヘッダーとして配置する場合に効果的です。

<Canvas of={NavbarStories.GlassEffect} />

## コンテンツの配置

複数の `Navbar.Content` コンポーネントを使用し、`justify` プロパティ（`start`, `center`, `end`）を組み合わせることで、柔軟なレイアウトが可能です。
