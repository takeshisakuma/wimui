# ChatUI

A comprehensive set of components for building chat interfaces with support for messages, avatars, timestamps, and interactive input.

## Components

The ChatUI module includes several sub-components:

<table>
  <thead>
    <tr>
      <th>Component</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>ChatContainer</strong></td>
      <td>The main wrapper for the chat interface</td>
    </tr>
    <tr>
      <td><strong>ChatMessageList</strong></td>
      <td>Scrollable container for messages</td>
    </tr>
    <tr>
      <td><strong>ChatMessage</strong></td>
      <td>Individual message bubble with support for avatars, timestamps, and sender names</td>
    </tr>
    <tr>
      <td><strong>ChatInput</strong></td>
      <td>Input field with send button for composing messages</td>
    </tr>
    <tr>
      <td><strong>ChatAvatar</strong></td>
      <td>Avatar component with image support and fallback text</td>
    </tr>
  </tbody>
</table>

## Best Practices

<table>
  <thead>
    <tr>
      <th>Best Practice</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>Position Messages</strong></td>
      <td>Use `position="left"` for received messages and `position="right"` for sent messages</td>
    </tr>
    <tr>
      <td><strong>Show Avatars</strong></td>
      <td>Display avatars for multi-user conversations to help identify senders</td>
    </tr>
    <tr>
      <td><strong>Include Timestamps</strong></td>
      <td>Help users understand message chronology</td>
    </tr>
    <tr>
      <td><strong>Auto-scroll</strong></td>
      <td>Implement auto-scroll to newest messages for better UX</td>
    </tr>
    <tr>
      <td><strong>Message Grouping</strong></td>
      <td>Group consecutive messages from the same sender</td>
    </tr>
    <tr>
      <td><strong>Loading States</strong></td>
      <td>Show typing indicators or loading states when appropriate</td>
    </tr>
    <tr>
      <td><strong>Accessibility</strong></td>
      <td>Ensure proper ARIA labels and keyboard navigation</td>
    </tr>
  </tbody>
</table>

## Accessibility

- Input field is keyboard accessible
- Send button can be triggered with Enter key
- Proper semantic HTML structure
- Avatar images include alt text
- Color contrast meets WCAG standards
