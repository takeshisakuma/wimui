# FieldError

`FieldError` コンポーネントは、入力フォームのバリデーションエラーをユーザーに通知するために使用します。

## 特徴
- **視認性:** `Paragraph` コンポーネントの `color="error"` と `size="small"` を標準で使用し、重要度を伝えます。
- **アクセシビリティ:** `role="alert"` と `aria-live="polite"` を備えており、スクリーンリーダーを使用するユーザーにエラーの発生を即座に伝えます。
- **一貫性:** フォーム全体でエラー表示のスタイルを統一できます。

## 使い方

```tsx
import { FieldError } from "./FieldError";

<FieldError content="メールアドレスの形式が正しくありません。" />
```
