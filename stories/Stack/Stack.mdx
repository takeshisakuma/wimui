`Stack` コンポーネントは、要素を垂直または水平に並べ、要素間の間隔を管理するために使用されます。

## 使い方

```tsx
import { Stack } from "@/components/Stack/Stack";
import { Box } from "../Box/Box";

const Demo = () => (
  <Stack gap="md">
    <Box>Item 1</Box>
    <Box>Item 2</Box>
    <Box>Item 3</Box>
  </Stack>
);
```

## 特徴

<table>
  <thead>
    <tr>
      <th>特徴</th>
      <th>説明</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>方向のリセット</strong></td>
      <td><code>direction</code> プロパティで <code>row</code> または <code>column</code> を指定できます（デフォルトは <code>column</code>）。</td>
    </tr>
    <tr>
      <td><strong>強力なスペース管理</strong></td>
      <td><code>gap</code> プロパティに <code>xs</code>, <code>sm</code>, <code>md</code> などのトークン、または数値を指定して間隔を調整できます。</td>
    </tr>
    <tr>
      <td><strong>Flexboxベース</strong></td>
      <td>内部で <code>Flex</code> コンポーネントを使用しているため、<code>align</code> や <code>justify</code> などの Flexbox プロパティも使用可能です。</td>
    </tr>
  </tbody>
</table>
