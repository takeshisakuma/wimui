# 入力系コンポーネントの使い分け

ユーザーからの入力を受け取る際、入力内容の種類や量に基づいて適切なコンポーネントを選択します。

## 入力タイプ比較

<table> 
  <thead>
    <tr>
      <th>コンポーネント</th>
      <th>入力内容</th>
      <th>用途</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>Input (Text/Email/Tel)</strong></td>
      <td>1行のテキスト</td>
      <td>名前、メールアドレス、電話番号など。</td>
    </tr>
    <tr>
      <td><strong>PasswordInput</strong></td>
      <td>パスワード</td>
      <td>表示/非表示を切り替えたい認証情報。</td>
    </tr>
    <tr>
      <td><strong>SearchInput</strong></td>
      <td>検索クエリ</td>
      <td>検索フォーム、リアルタイムフィルタ。</td>
    </tr>
    <tr>
      <td><strong>NumberInput</strong></td>
      <td>数値</td>
      <td>数量、価格、ステップ調整が必要な設定値。</td>
    </tr>
    <tr>
      <td><strong>Textarea</strong></td>
      <td>複数行のテキスト</td>
      <td>メッセージ、備考、長い説明文。</td>
    </tr>
    <tr>
      <td><strong>InputGroup</strong></td>
      <td>アイコン/単位付き</td>
      <td>通貨入力、検索バー、URLプレフィックス付き入力。</td>
    </tr>
    <tr>
      <td><strong>OtpInput</strong></td>
      <td>数字コード</td>
      <td>2要素認証コード、確認コード。</td>
    </tr>
    <tr>
      <td><strong>InputMask</strong></td>
      <td>フォーマット制限</td>
      <td>クレカ番号、日付、電話番号などの定型入力。</td>
    </tr>
    <tr>
      <td><strong>Mentions</strong></td>
      <td>メンション</td>
      <td>@によるユーザー指定、#によるタグ指定。</td>
    </tr>
    <tr>
      <td><strong>Rating</strong></td>
      <td>評価</td>
      <td>星による5段階評価などのユーザーフィードバック。</td>
    </tr>
    <tr>
      <td><strong>FileUpload / Dropzone</strong></td>
      <td>ファイル</td>
      <td>プロフィール画像、実録、ドキュメントのアップロード。</td>
    </tr>
  </tbody>
</table>

## 入力補助とバリデーション

<table>
  <thead>
    <tr>
      <th>項目</th>
      <th>ガイドライン</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>Label</strong></td>
      <td>すべての入力には明確なラベルが必要です。プレースホルダーをラベルの代わりとして使用しないようにしてください。</td>
    </tr>
    <tr>
      <td><strong>FieldError</strong></td>
      <td>入力エラーは、エラーが発生している入力フィールドのすぐ近くに表示します。</td>
    </tr>
    <tr>
      <td><strong>Input Group</strong></td>
      <td>
        <ul>
          <li>単位が必要な場合（例：円）は、入力欄の右側に単位を表示します。</li>
          <li>検索など特定の機能を持つ場合は、アイコンを表示して視認性を高めます。</li>
        </ul>
      </td>
    </tr>
  </tbody>
</table>
