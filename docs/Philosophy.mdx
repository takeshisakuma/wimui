# Philosophy

## 手触りのあるUIを追求

そのために以下の点に注力して設計・開発を行います。

- **アクセシビリティ**
  - すべてのユーザーが利用できるように、キーボード操作に対応します。
  - WAI-ARIAを活用し、スクリーンリーダー等の支援技術のサポートを強化します。
- **インタラクション**
  - ユーザーの操作に対して、期待通りのフィードバックを迅速かつ自然に行い、「手触りのある」心地よい体験を提供します。
