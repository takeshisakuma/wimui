# ローディング状態の使い分け

データの取得中や処理中であることをユーザーに伝え、待機時間のストレスを軽減します。

## ローディング形式の選択

  <tbody>

        <strong>Loader</strong>
      </td>

    </tr>
    <tr>
      <td>
        <strong>Spinner</strong>
      </td>
      <td>回転アイコン</td>

    </tr>
    <tr>
      <td>

      </td>
