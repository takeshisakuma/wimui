<h2>SCSSでの利用方法 (Mixins)</h2>

<p>これらのトークンは、<code>src/tokens/_breakpoints.scss</code> 内でマップ変数 <code>$breakpoints</code> として定義されており、メディアクエリを簡単に書くための Mixin も用意されています。</p>

<h3><code>@include up($breakpoint)</code></h3>
<p>指定したブレイクポイント<strong>以上 (min-width)</strong>でスタイルを適用する「モバイルファースト」なレイアウトで使用します。</p>

```scss
@use "../../tokens/breakpoints" as *;

.my-component {
  // デフォルト（スマートフォン等の小さい画面向け）
  display: block;

  @include up(md) {
    // 画面幅が 768px 以上の時のスタイル
    display: flex;
  }
}
```

<h3><code>@include down($breakpoint)</code></h3>
<p>指定したブレイクポイント<strong>以下 (max-width)</strong>でのみスタイルを適用する際に使用します。</p>

```scss
@use "../../tokens/breakpoints" as *;

.my-button {
  padding: 16px;

  @include down(sm) {
    // 画面幅が 576px 以下の時にだけパディングを小さくする
    padding: 8px;
  }
}
```
