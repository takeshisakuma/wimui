<h3>`@include up($breakpoint)`</h3>
<p dangerouslySetInnerHTML={{ __html: "<T k='token.bp_up_desc' />" }} />

```scss
@use "../../tokens/breakpoints" as *;

.my-component {
  // Default (for small screens like smartphones)
  display: block;

  @include up(md) {
    // Style for screen width 768px and above
    display: flex;
  }
}
```

<h3>`@include down($breakpoint)`</h3>
<p dangerouslySetInnerHTML={{ __html: "<T k='token.bp_down_desc' />" }} />

```scss
@use "../../tokens/breakpoints" as *;

.my-button {
  padding: 16px;

  @include down(sm) {
    // Reduce padding only when screen width is 576px or less
    padding: 8px;
  }
}
```
