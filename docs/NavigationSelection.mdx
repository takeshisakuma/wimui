# ナビゲーション要素の使い分け

ユーザーがアプリケーション内やコンテンツ内で移動したり、表示を切り替えたりするための手段を選択します。

## ナビゲーションタイプ比較

<table>
  <thead>
    <tr>
      <th>コンポーネント</th>
      <th>配置</th>
      <th>階層レベル</th>
      <th>主な用途</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>
        <strong>Tabs</strong>
      </td>
      <td>コンテンツ内</td>
      <td>中〜下</td>
      <td>同一コンテキスト内での表示切り替え。</td>
    </tr>
    <tr>
      <td>
        <strong>TabNavigation</strong>
      </td>
      <td>ページ要素</td>
      <td>中</td>
      <td>URLと連動した表示切り替え。</td>
    </tr>
    <tr>
      <td>
        <strong>Breadcrumb</strong>
      </td>
      <td>コンテンツ上部</td>
      <td>全レベル</td>
      <td>現在地の把握、上位階層への素早い復帰。</td>
    </tr>
    <tr>
      <td>
        <strong>Pagination</strong>
      </td>
      <td>リスト下部</td>
      <td>リスト</td>
      <td>大量データのページ分割移動。</td>
    </tr>
    <tr>
      <td>
        <strong>Stepper</strong>
      </td>
      <td>画面中央/上部</td>
      <td>手順</td>
      <td>順序が決まったプロセス（登録、決済等）。</td>
    </tr>
    <tr>
      <td>
        <strong>Anchor</strong>
      </td>
      <td>コンテンツ内</td>
      <td>ページ内</td>
      <td>長いページでの特定のセクションへのショートカット。</td>
    </tr>
    <tr>
      <td>
        <strong>HamburgerMenu</strong>
      </td>
      <td>ヘッダー内等</td>
      <td>補助</td>
      <td>メニュー内容が多い場合のモバイル用集約。</td>
    </tr>
  </tbody>
</table>
