<p>
  <T k="guide.lvb_antipattern_text" />
</p>

```jsx
// Bad: Button behaving as link
<button onClick={() => window.location.href = '/about'}>About (Bad)</button>

// Bad: Anchor behaving as button
<a href="#" onClick={saveData}>Save (Bad)</a>
```
