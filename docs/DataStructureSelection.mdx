# データ構造の使い分け

大量のデータや複雑な親子関係、時系列データを整理して表示します。

## 比較・ユースケースガイド

<table>
  <thead>
    <tr>
      <th>コンポーネント</th>
      <th>構造・形式</th>
      <th>特徴・できること</th>
      <th>ユースケース</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>Table</strong></td>
      <td>格子状<br/>(Row/Column)</td>
      <td>軽量でシンプルな表の生成。データの比較・分析、ソート、フィルタリングが必要な場面に最適。</td>
      <td>ユーザーの一覧表示、設定項目のリスト、閲覧を主な目的とし、一度に表示するデータ件数が限定的な画面。</td>
    </tr>
    <tr>
      <td><strong>DataGrid</strong></td>
      <td>高度な格子状</td>
      <td>大量データの仮想化表示（仮想スクロール）、列の固定、セルの直接編集など、スプレッドシートのような高度でリッチな機能を提供。</td>
      <td>数千件〜数万件のレコードを素早くスクロールさせる必要がある画面や、情報量の多い多機能な業務データ画面。</td>
    </tr>
    <tr>
      <td><strong>DescriptionList</strong></td>
      <td>キー・値ペア</td>
      <td>項目名（キー）とその詳細（値）のペアをきれいに成形して一覧表示する。</td>
      <td>ユーザープロフィールの詳細画面や、画面幅を問わずにレスポンシブな配置が必要な商品の仕様スペック一覧。</td>
    </tr>
    <tr>
      <td><strong>List</strong></td>
      <td>垂直一列</td>
      <td>シンプルなリスト項目の縦並び表示。行単位の独立性を保ったモバイル重視のレイアウトに適している。</td>
      <td>モバイル端末でのアイテム一覧、通知フィードの表示など。</td>
    </tr>
    <tr>
      <td><strong>TreeView</strong></td>
      <td>階層・ツリー</td>
      <td>ディレクトリ構造などの階層（親子関係）を持つデータの折りたたみ・展開と視覚的表現。</td>
      <td>ファイラーのようなディレクトリ構成の表示、組織図、多階層にわたるカテゴリツリーの表示。</td>
    </tr>
    <tr>
      <td><strong>Timeline</strong></td>
      <td>時系列</td>
      <td>更新履歴や進捗の流れなど、時間軸に沿って連続するイベントを視覚的なフローとして表現する。</td>
      <td>Eコマースでの商品の配送状況、ワークフローの進捗状況、システムの実行履歴やユーザーのアクティビティログ表示。</td>
    </tr>
    <tr>
      <td><strong>Stats</strong></td>
      <td>数値・指標</td>
      <td>KPIなど主要な数値データ（指標）を視覚的に強調表示し、増減などのトレンドも併せて表現する。</td>
      <td>アナリティクスダッシュボード画面でのトップ表示、今日の売上高・新規ユーザー数といった重要KPIのハイライト。</td>
    </tr>
    <tr>
      <td><strong>ChatUI</strong></td>
      <td>メッセージ形式</td>
      <td>メッセージ一覧、入力、アバター等のチャット専用パーツを提供。リアルタイムな対話 UI を容易に構築。</td>
      <td>サポートチャット、AIチャット、カスタマー等とのメッセージング画面に。</td>
    </tr>
  </tbody>
</table>
