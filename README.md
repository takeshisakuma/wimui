# wimui

## npm

### 作業時

npm run storybook

### npmのパッケージのバージョン確認

npm outdated

### 品質確認

npm run stylelint // storiesフォルダ内のscssの品質確認
npm run lint // storiesフォルダ内のjs,tsの品質確認
npm run test // コンポーネント単体テスト (*.test.tsx) の実行

### 整形

npm run format

### github pagesへのデプロイ

npm run deploy
